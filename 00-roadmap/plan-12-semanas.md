# Plan de 12 semanas (calmado y constante)

> Ritmo sugerido: 1h diaria, flexible. Si una semana no se completa, se arrastra sin estrés.

## Semana 1-4: Fundamentos Go
- S1: instalación mental de Go (package main, func main, variables, tipos)
- S2: control de flujo (if, for), funciones, múltiples returns
- S3: slices, maps, structs
- S4: métodos, punteros, interfaces básicas

## Semana 5-8: DSA básico
- S5: complejidad temporal/espacial (Big-O intuitivo)
- S6: arrays/slices y linked lists
- S7: stacks, queues, hash maps
- S8: recursión + problemas simples

## Semana 9-10: Patrones de algoritmos
- S9: two pointers + sliding window
- S10: binary search pattern + ordenamientos

## Semana 11-12: Integración en proyectos
- S11: mini CLI (tareas o notas)
- S12: mini API + repaso general

## Definición de “tema dominado”
Un tema está “dominado” si podés:
1. explicarlo en 5 líneas,
2. resolver 2 ejercicios sin mirar,
3. detectar un error típico.
