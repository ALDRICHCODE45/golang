# Clase 1 — Hola Mundo + variables

## Objetivo
Entender la estructura mínima de un programa Go y declarar variables simples.

## Práctica guiada
1. Crear `main.go`.
2. Imprimir tu nombre y tu objetivo con Go.
3. Declarar variables: `nombre`, `edad`, `aprendiendoGo`.
4. Mostrar todo con `fmt.Printf`.

## Práctica solo
- Crear un programa que imprima:
  - nombre,
  - ciudad,
  - cantidad de horas semanales para estudiar.

## Checklist
- [ ] Entiendo para qué sirve `package main`.
- [ ] Entiendo por qué existe `func main()`.
- [ ] Diferencio `:=` de `var`.
