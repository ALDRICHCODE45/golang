# Go + DSA Journey (Hobby)

Este repo es tu bitácora de aprendizaje en Go y estructuras de datos/algoritmos.

## Enfoque
- 20% teoría, 80% práctica.
- Ritmo: **1 hora por día** cuando se pueda (sin culpa si algún día no se llega).
- Objetivo: entender fundamentos, no copiar código.

## Cómo usar esta carpeta
1. Elegí el módulo actual en `00-roadmap/plan-12-semanas.md`.
2. Hacé los ejercicios del tema.
3. Documentá lo aprendido en `99-journal/diario.md`.
4. Registrá errores en `99-journal/errores-comunes.md`.

## Regla de oro
Si no podés explicarlo con tus palabras, todavía no lo entendiste.
